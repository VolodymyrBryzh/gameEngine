# Survival Engine SDK

Цей SDK призначений для розширення можливостей двигуна Survival Engine шляхом створення власних моделей, ігрової логіки та анімацій.

## Структура
- `include/`: Заголовні файли з інтерфейсами.
- `examples/`: Приклади реалізації.

## Як почати
Для створення нової сутності успадкуйте клас `IEntity` з `survival_sdk.h` та реалізуйте необхідні методи.
